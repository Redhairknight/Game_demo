# Nikki Survivors - 集成测试检查清单

> 以下测试需要在 Godot 编辑器中手动执行，确认游戏系统间的交互正常。

## 玩家控制

- [ ] 玩家移动流畅（WASD 8方向）
- [ ] 对角线移动速度标准化（不超过直线速度）
- [ ] 移动动画方向正确切换
- [ ] 站立时回到待机动画
- [ ] 玩家不会穿透地图边界

## 生命系统

- [ ] HP条被攻击时正确减少
- [ ] HP数值显示与条长度一致
- [ ] HP降到0触发 Game Over
- [ ] 受伤时有无敌帧闪烁
- [ ] 恢复道具正确增加HP

## 衣着系统

- [ ] 衣着条独立于HP
- [ ] 衣着耐久下降时立绘/精灵正确切换阶段
- [ ] 阶段0→1→2→3→4 视觉变化明显
- [ ] 极限形态（0%）触发特殊效果
- [ ] 衣着不影响基础HP
- [ ] 解放选项正确消耗衣着耐久

## 挑逗系统

- [ ] E键按住开始蓄力
- [ ] 蓄力UI指示器正确显示进度
- [ ] 蓄力期间角色不可移动
- [ ] 蓄力时被攻击中断并清零
- [ ] 释放时根据档位产生不同范围效果
- [ ] 满蓄力消耗1阶段衣着
- [ ] 挑逗范围内敌人正确受到效果（嘲讽/减速）
- [ ] 冷却时间内E键无反应

## 敌人系统

- [ ] 敌人追踪玩家（寻路不卡墙）
- [ ] 敌人接触玩家造成伤害
- [ ] 敌人死亡播放动画后消失
- [ ] 敌人死亡掉落经验宝石
- [ ] 精英怪有明显视觉区分
- [ ] 精英怪属性（HP、伤害）明显高于普通怪
- [ ] 敌人数量随时间增加
- [ ] 达到上限后不再生成新敌人
- [ ] 敌人不从屏幕内凭空出现

## 武器系统

- [ ] 武器自动攻击（无需手动操作）
- [ ] 武器冷却指示器正确显示
- [ ] 弹幕飞行方向正确（朝向最近敌人/固定方向）
- [ ] 弹幕击中敌人造成伤害
- [ ] 弹幕数量随等级增加
- [ ] 多武器同时独立运作

## 经验与升级

- [ ] 经验宝石被角色拾取范围自动吸引
- [ ] 经验条正确填充
- [ ] 达到阈值触发升级
- [ ] 升级面板3选1正常弹出
- [ ] 升级面板出现时游戏暂停
- [ ] 选择后面板关闭游戏恢复
- [ ] 选项包含正确的类型（武器/被动/解放/特殊）
- [ ] 选择武器升级后伤害/弹幕正确提升
- [ ] 选择被动后属性正确应用
- [ ] 经验拾取和升级

## 解放系统

- [ ] 解放选项在升级面板中正确出现
- [ ] 选择解放消耗对应衣着耐久
- [ ] 解放技能效果正确生效
- [ ] 衣着耐久不足时解放选项灰显/不可选

## 游戏流程

- [ ] 游戏15分钟计时器正确倒计/正计
- [ ] 计时器UI持续显示
- [ ] 15分钟结束触发BOSS/胜利条件
- [ ] Game Over触发和显示（HP归零）
- [ ] Game Over画面显示统计信息（存活时间、击杀数等）
- [ ] Game Over可返回主菜单或重试
- [ ] 胜利画面显示（15分钟存活成功）

## 性能

- [ ] 100+敌人同屏不卡顿（维持60FPS）
- [ ] 弹幕密集时无明显掉帧
- [ ] 经验宝石大量堆积无内存泄漏
- [ ] 长时间游戏（10分钟+）无性能退化

## 音效与UI

- [ ] 背景音乐正常播放
- [ ] 攻击/受伤/升级音效正确触发
- [ ] 暂停菜单正常工作
- [ ] HUD元素（HP、衣着、经验、计时器）始终可见
- [ ] 分辨率切换后UI适配正确

---

## 测试环境

- **引擎版本**: Godot 4.4
- **目标帧率**: 60 FPS
- **测试分辨率**: 1280x720（默认）, 1920x1080
- **输入设备**: 键盘 (WASD + E)
