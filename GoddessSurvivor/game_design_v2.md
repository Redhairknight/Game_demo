## 一、设计目标

| 维度 | 决策 |
|------|------|
| 核心玩法 | Vampire Survivors 类自动射击 + 美少女擦边像素风 |
| 目标受众 | Steam独立游戏玩家（可稍大胆，性感但俏皮） |
| 单局时长 | 10-15分钟（经典VS节奏） |
| 尺度定位 | 像素风降低激进感，"华丽碎裂"而非"撕裂" |
| 叙事包装 | 暴露 = 觉醒/解放力量，角色从害羞→自信 |

## 二、核心循环（升级版）

```
移动闪避 → 自动攻击击杀 → 掉落经验/道具 → 拾取升级 → 选择强化
    ↕                                                    ↕
E键挑逗（主动消耗衣着）                       升级时出现"解放选项"
    ↓                                                    ↓
蓄力释放AOE清屏                              衣着-1阶段 → 角色专属加成
    ↓                                                    ↓
   └──────── 武器Lv5 + 暴露阶段3 = 协同觉醒！ ─────────┘
                              ↓
                       极限形态30秒爆发
                              ↓
                       换装宝箱3选1（新循环）
```

### 核心设计原则
1. **暴露完全由玩家主动控制**——没有任何"被迫脱衣"机制
2. **暴露 = 强力**——主动追求而非被动惩罚
3. **双build路线协同**——武器线和暴露线独立可玩，协同最强
4. **修复 = 换装**——不是无聊的"修好"而是有趣的"换一套"

## 三、双条系统

### 3.1 HP条（红色·生存线）

| 要素 | 说明 |
|------|------|
| 减少方式 | 被敌人攻击 |
| 恢复方式 | 拾取HP道具、升级时少量恢复 |
| 归零后果 | Game Over |
| 与衣着关系 | **完全独立，互不影响** |

### 3.2 衣着耐久度条（粉色·擦边线）

| 要素 | 说明 |
|------|------|
| 初始状态 | 满值（对应衣着阶段0：完好） |
| 减少方式 | ① 升级选"解放选项" ② E键挑逗消耗 ③ 特殊事件选择 |
| 恢复方式 | ① 换装宝箱（3选1切换服装） ② 升级时"强化服装"选项 ③ 角色专属（如雪每30秒+1） |
| 归零后果 | 进入"极限形态"（全暴露最强态，限时30秒后强制修复到阶段2） |
| 核心定位 | **可消耗的战略资源** |

### 3.3 衣着阶段表

| 耐久度 | 阶段 | 视觉状态 | 效果 |
|--------|------|----------|------|
| 100%~81% | 0 | 完好 | 无加成 |
| 80%~61% | 1 | 轻微损坏 | 角色专属加成Lv1 |
| 60%~41% | 2 | 中度损坏 | 角色专属加成Lv2 |
| 40%~21% | 3 | 严重损坏 | 角色专属加成Lv3 + 可触发协同觉醒 |
| 20%~1% | 4 | 极限状态 | 角色专属加成Lv4（最强被动） |
| 0% | 极限形态 | 全暴露 | 全属性暴涨30秒 + 无敌 |

## 四、E键挑逗系统

### 4.1 基础机制

```
按住E键：
├── 0.5秒后进入"挑逗姿态"
├── 持续期间：
│   ├── 角色无法移动（站桩）
│   ├── 每秒消耗1格衣着耐久度
│   ├── 周围敌人被"嘲讽"→ 向你聚集但移速-50%
│   └── 蓄力条逐渐充满
├── 松开E = 释放"魅力冲击波"：
│   ├── 蓄力1秒：小范围击退
│   ├── 蓄力2秒：中范围伤害 + 减速3秒
│   └── 蓄力3秒（满蓄）：大范围秒杀 + 衣着直接-1阶段
└── 被击中 = 强制打断，蓄力清零
```

### 4.2 高级操作（skill expression）

| 技巧 | 条件 | 效果 |
|------|------|------|
| 完美闪避挑逗 | 敌人攻击命中瞬间按E | 自动闪避 + 立即满蓄力释放 |
| 连续挑逗 | 满蓄力释放后0.3秒内再按E | 第二次蓄力速度×2 |
| 三连pose | 连续完成3次满蓄力释放 | 成就 + 额外掉落奖励 |

### 4.3 设计意图

- 把"擦边"变成可操作的技巧表达
- 聚怪→蓄力→释放 = 类似割草游戏的"大招"循环
- 有风险（站桩可能被打断）有回报（清屏AOE）
- "挑逗"主题让擦边不再被动——角色在自信地展示自己

## 五、换装系统（修复 = 换装）

### 5.1 核心概念

- 游戏中不存在传统的"修复"
- 拾取「换装宝箱」时，出现3选1换装选择
- 不同服装有不同的base属性和损坏视觉
- 一局内可能穿过多套衣服

### 5.2 换装宝箱出现频率

- 约每3-4分钟出现1次（一局约出现3-4个）
- Boss击杀后必定掉落
- 玩家也可以选择"不换"（保持当前服装，获得+1阶段耐久恢复）

### 5.3 服装设计示例（铃·Rin）

| 服装 | Base属性 | 起始阶段 | 损坏路线 |
|------|----------|----------|----------|
| 📚 学院制服 | 均衡 | 0 | 裙→上衣→内衣 |
| 👙 学校泳装 | 攻速+30% | 2（起始半裸） | 泳装top→底部→极限 |
| 🎃 啦啦队服 | 移速+25% | 0 | 裙摆→腹部→胸部 |
| 🌸 巫女见习服 | 经验+40% | 0 | 袖口→下摆→上身 |

### 5.4 换装决策点

- 当前暴露阶段高（加成强）→ 换装会重置加成，但获得新base属性
- 当前暴露阶段低（刚开始）→ 换装几乎无损失，可以挑选最佳属性
- "不换"选项 → 恢复1阶段耐久，保持当前路线
- 这构成了局内的**中期策略决策**

## 六、Build路线：武器 × 暴露 协同系统

### 6.1 三种Build路线

| 路线 | 描述 | 优势 | 劣势 |
|------|------|------|------|
| 纯武器Build | 专注升级武器+被动，达成武器进化 | 安全稳定 | 缺少爆发力 |
| 纯暴露Build | 推进衣着损坏，利用角色专属加成 | 角色能力强 | 武器弱 |
| 协同Build | 武器+暴露同时推进，达成"协同觉醒" | 最强终极形态 | 条件苛刻 |

## 七、角色差异化暴露设计

### 🎀 铃（Rin）— 学院制服 → 「甜心轰炸」路线

| 阶段 | 视觉 | 加成 |
|------|------|------|
| 1 | 裙摆破损 | 弹幕环发射频率+15% |
| 2 | 上衣肩部破损 | 弹幕环获得"追踪"属性 |
| 3 | 制服大面积碎裂 | 周围自动吸取经验范围×2 |
| 4 | 极限（内衣状态） | 弹幕变心形，"魅惑"敌人互攻 |

**设计意图**：弹幕密度型，暴露让弹幕从"多"变成"智能"

### ⚔️ 凛（Lin）— 骑士甲胄 → 「逆鳞觉醒」路线

| 阶段 | 视觉 | 加成 |
|------|------|------|
| 1 | 肩甲脱落 | 受伤时反弹30%伤害 |
| 2 | 胸甲裂开 | 获得"格挡"（每5秒免疫一次攻击） |
| 3 | 甲胄大片剥落 | 近身敌人持续受到灼烧光环伤害 |
| 4 | 极限（锁骨以上覆甲） | "觉醒骑士"无敌冲锋，碰撞即杀 |

**设计意图**：重甲脱甲=解放力量，叙事天然合理

### 🌙 零（Rei）— 魔法礼裙 → 「以太共振」路线

| 阶段 | 视觉 | 加成 |

| 4 | 极限（纯魔力形态） | 全身电弧，触碰即死区域 |

### 🔥 焰（Homura）— 暗黑女仆 → 「爆裂侍从」路线

## 八、升级选择系统

### 8.1 升级时出现的选项类型

每次升级出现 3选1（偶尔4选1作为奖励），选项池包含：

| 类型 | 概率 | 示例 |
|------|------|------|
| 🔫 武器升级 | 40% | 弹幕环 Lv3→Lv4 |
| 📦 被动道具 | 30% | 疾风鞋 Lv1 |
| 👗 解放选项 | 20% | "解放上衣：攻速+20%，衣着-1阶段" |
| ✨ 特殊选项 | 10% | "强化服装+1阶段" / "神秘换装" |

### 8.2 "解放选项"示例

```
👗 解放·上衣
  效果：攻速永久+20%
  代价：衣着耐久-1阶段
  
👗 解放·短裙
  效果：移速永久+15%
  代价：衣着耐久-1阶段
  
👗 解放·觉醒
  效果：当前武器伤害+30%
  代价：衣着耐久-2阶段（跳过一级）
```

## 九、跨局系统：亲密度养成

### 9.1 亲密度积累

| 来源 | 数值 |
|------|------|
| 每局基础完成 | +5 |
| 达到暴露阶段2 | +3 |
| 达到暴露阶段3 | +5 |
| 达到暴露阶段4 | +8 |
| 触发极限形态 | +10 |
| 触发协同觉醒 | +15 |
| 使用挑逗击杀50+敌人 | +5 |

### 9.2 亲密度解锁内容

| 亲密度 | 解锁 |
|--------|------|
| 10 | 角色语音包1（战斗台词） |
| 25 | 角色专属服装1 |
| 50 | 角色互动动画（摸头/击掌） |
| 75 | 角色语音包2（暴露台词） |
| 100 | 隐藏角色「焰·Homura」解锁 |
| 150 | "约定形态"（最终造型+专属超强武器） |

### 9.3 亲密度特点
- 所有角色独立计算
- 永不重置
- 提供长期目标和角色养成感
- "我想看完整个角色的故事" = 核心留存驱动

## 十二、视觉方向（像素风）

### 风格参考
- 像素精度：角色 48×48 或 64×64
- 色彩：柔和明亮的日系色板
- 损坏特效："华丽碎裂"——布片带闪光粒子飞散
- 暴露遮挡：光芒/魔法纹/战斗特效半遮，擦边但有设计感
- 角色表情：从"害羞惊讶"到"自信微笑"的渐变

### 关键视觉要素
| 要素 | 处理方式 |
|------|----------|
| 衣着碎裂 | 闪光粒子+布片飘散动画 |
| 暴露部分 | 魔法纹/光芒半遮，暗示而非直接 |
| 挑逗姿态 | 俏皮可爱的pose，配合心形/星形特效 |
| 极限形态 | 全身发光+能量溢出视觉，力量感优先于色情感 |
| 换装过渡 | 短暂的闪白+新服装碎片聚合的正向动画 |
