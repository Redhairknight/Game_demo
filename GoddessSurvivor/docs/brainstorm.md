## 一、当前核心循环

```
移动闪避 → 自动攻击击杀 → 掉落经验宝石 → 拾取升级 → 选择强化 → 更强敌潮 → 循环
                                                          ↕
                                              受伤 → 衣着损坏 → 视觉反馈
```

### 当前问题

- "衣着损坏"只是受伤的**被动结果**，玩家没有主动参与感
- 擦边元素是"被迫看到的"而非"主动追求的"，削弱了驱动力
- 想看更多暴露 = 必须濒死 = 很快 Game Over，代价过高

## 三、擦边驱动机制候选方案

### 方案A：「解放系统」— 损坏 = 角色增强

| 阶段 | 暴露程度 | 能力加成 |
|------|----------|----------|
| 1 | 轻微 | 攻速+10% |
| 2 | 中度 | 移速+15%，解锁闪避动画 |
| 3 | 严重 | 攻击力+25%，范围+20% |
| 4 | 极限 | 全属性暴涨，但HP不再恢复，进入"觉醒态" |

**核心决策**：修复衣着（安全但弱） vs 保持暴露（危险但强）

---

### 方案B：「魅力值」— 独立擦边货币

```
新增「魅力值」资源条:
- 衣着每损坏一阶段 → 魅力值持续产生
- 魅力值兑换: 稀有武器进化、限定造型、强力道具
- 阶段越高，产出越快
```

**核心决策**：花魅力值升级 vs 存魅力值等大招

---

### 方案C：「观众系统」— 虚拟观众互动

```
屏幕边缘有"观众"弹幕/图标:
- 暴露越多 → 观众越多 → buff越强
- 观众投掷: 金币、回复、武器碎片
- 保持完好 → 观众离开 → buff消失
```

**核心决策**：为观众"表演"获得即时收益

---

### 方案D：「亲密度」— 跨局情感积累

```
- 新增「亲密度」进度条，独立于HP
- 每次进入新损坏阶段 → 解锁角色语音/表情/互动
- 亲密度满 → 解锁"约定形态"（最终造型+专属超强武器）
- 跨局继承，永不重置
```

**核心决策**：长期养成驱动 + 情感包装

---

### 方案E：「逆铠甲」— 脱得越多越硬

```
每损坏一阶段:
- 触发"气场展开": 身周0.5秒无敌护盾
- 永久获得闪避率+8%（角色做回避动作展示曲线）
- 解锁"魅影"分身（残影吸收伤害）

极限阶段:
- 变为"以太形态"，受伤减半
- 被修复 → 失去所有加成，从头积累
```

**核心决策**：颠覆直觉（脱衣=变强），害怕被修复是有趣的情感

---

### 方案F：「换装射击」— 碎片变弹幕

```
每次衣着损坏:
- 碎裂布料变成旋转弹幕
- 损坏越多 → 布片弹幕层数越多 → 近身DPS爆炸

进化终极:
- 满5阶段 → 布片聚合为"绝对领域"力场
- 持续伤害，但角色几乎全裸
```

**核心决策**：暴露和战斗力直接可视化挂钩

---

### 方案G：「挑逗系统」— 主动展示操作

```
新增「挑逗」键（E键）:
- 角色做pose → 周围敌人嘲讽/减速3秒
- 期间无法移动，衣着主动损坏
- 阶段越高，挑逗效果越强

高级操作:
- "挑逗取消": 挑逗中瞬间闪避 = 高操作收益
- "完美挑逗": 敌人命中瞬间挑逗 → 反杀大范围伤害
```

**核心决策**：把擦边变成 skill expression，有操作上限

---

### 方案H：「温度条」— 衣着 = 散热

```
- 攻击产生热量，热量满 → 武器过载停火3秒
- 衣着损坏 = 散热面积大 = 过热慢 = 持续输出久
- 玩家主动"轻装上阵"追求高DPS

视觉:
- 角色身上发光/蒸汽特效
- 暴露皮肤有散热纹路/魔法阵发光
```

**核心决策**：合理化"脱衣=变强"的内在逻辑

## 四、推荐组合方案

```
核心系统 = 方案E（逆铠甲/脱越多越强）+ 方案F（碎片变弹幕）
辅助系统 = 方案D（亲密度跨局积累）
操作亮点 = 方案G（挑逗）简化版

融合体验:
1. 受伤 → 衣着碎裂 → 碎片变旋转弹幕（战力直接提升）
2. 损坏越深 → 闪避率提高 + 弹幕层数增加（主动追求暴露）
3. 按E主动挑逗 → 范围减速 + 加速损坏（技巧表达）
4. 每局暴露时间累积"亲密度" → 跨局解锁新角色/造型
5. 修复道具变有趣: 不是"要不要修"而是"修到哪个最优层数"
```
